# ViralForge AI 🎬

ViralForge AI is an open-source tool that helps creators turn ideas or transcripts into viral short-form video scripts (TikTok, YouTube Shorts, Reels).

## 🚀 Features

* Generate high-retention hooks (first 3 seconds)
* Convert topics into 60-second viral scripts
* Create captions and engagement-focused endings

## 🧠 How It Works

1. Input a topic or transcript
2. AI generates a structured viral script:

   * Hook
   * Story
   * Call to Action

## 💡 Example Output

HOOK:
Wait... this will change how you see success forever.

SCRIPT:
Most people think success is about hard work...
But here's the truth...

CTA:
Follow for more.

## 🛠️ Installation

```bash
git clone https://github.com/mindmastery101/viralforge-ai-open-source
cd viralforge-ai-open-source
python app.py
```
